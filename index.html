How to learn JavaScript

If a person already knows how to write a computer program, it should be relatively easy to pick another language such as the Javascript language. In this page, we will try to examine how people with some background in computer science can pick up the skills needed for web development. Javascript is typically used in web development, so it makes sense to learn html and css first if one does not know or remember from before.

Setting Up Your Website locally and on Internet

You may be wondering how to build your website. It is really simple if you are just building it just for your local machine. You just have to understand the structure of your file system and how to navigate through your file system. If you want to build a website on the Internet for free, you can do it on github to publish your webpage to the Internet. Of course, if you pay out of pocket, you can pay InterServer or Hostinger to have even more powerful features for your website. Here is a series of videos by the Mike Zamansky.
